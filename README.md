### Tech

* [node.js](https://nodejs.org/en/)
* [Gulp](http://gulpjs.com/)
* [jQuery](http://jquery.com/) 

### Initialize

- Make sure Gulp is installed globally 
```sh
$ npm i -g gulp
```

- Clone this repo
```sh
$ git clone [https://github.com/weavler/fullstack-app.git]
$ cd fullstack-app
```

- Install modules
```sh
$ npm install
```

- Run the default gulp task
```sh
$ gulp
```
